# CRM-clients-php7
composer / php7 / doctrine / Smarty / Jquery / Mysql / Docker

## Installation

```bash
git clone https://github.com/afimpel/CRM-clients-php7.git 
cd CRM-clients-php7/
docker-compose up -d
docker exec -w /var/www/html php7apache composer install
```

## USO

```bash
firefox http://localhost/
```
